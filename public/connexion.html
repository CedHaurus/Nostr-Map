Pour ajouter votre profil Nostr à l'annuaire

Connectez-vous avec votre compte Nostr — votre profil sera automatiquement importé et visible dans l'annuaire.

⚡

Se connecter avec Nostr

Nostr est un protocole de réseau social décentralisé. Votre identité vous appartient entièrement — aucun mot de passe, aucun serveur central.

Pour accéder à votre profil Nostr Map, gérer vos liens sociaux vérifiés ou proposer un utilisateur à l'annuaire, vous avez besoin d'un compte Nostr et d'une extension installée dans votre navigateur.
1

Installer une extension Nostr

L'extension stocke votre clé privée localement dans votre navigateur. Elle signe les requêtes à votre place sans jamais transmettre votre clé à un site externe.

nos2x-fox — Firefox

Firefox
1

Installez nos2x-fox depuis le bouton ci-dessus, puis cliquez sur son icône dans la barre d'outils Firefox et ouvrez les Préférences.

2

Dans la section Profile, cliquez sur New pour créer un nouveau profil, ou sur Import si vous avez déjà un compte Nostr.

Section Profile : New pour créer, Import pour importer une clé existante
La section Keys affiche le champ Private key. Collez-y votre clé nsec… ou cliquez Generate pour en créer une nouvelle.
3

Cliquez sur Save key pour enregistrer. Vous pouvez activer la PIN Protection pour sécuriser vos clés avec un code.

Cliquez Save key, puis activez PIN Protection si vous le souhaitez

nos2x — Chrome / Edge

Chrome / Edge
1

Installez nos2x depuis le bouton ci-dessus, puis cliquez sur son icône dans la barre d'outils pour ouvrir les options.

2

Collez votre clé privée (nsec…) dans le champ private key, puis cliquez sur save.

Collez votre nsec… dans le champ private key et cliquez save
Les boutons browse your profile et edit your profile vous redirigent vers nosta.me pour consulter et modifier votre profil Nostr.
2

Vous n'avez pas encore de compte Nostr ?

Rendez-vous sur nostr.fr — la meilleure ressource en français pour apprendre à utiliser Nostr, créer son identité et découvrir les applications disponibles.

Une fois votre compte créé, revenez ici et importez votre clé privée dans l'extension (étape 1).

Apprendre à utiliser Nostr sur nostr.fr
3

Se connecter sur Nostr Map

Extension installée et configurée ? Cliquez sur le bouton ci-dessous. L'extension vous demandera d'approuver la connexion — cliquez Autoriser.

Votre profil Nostr existant est automatiquement importé sur Nostr Map.

Votre sécurité

  • Votre clé privée ne quitte jamais votre appareil
  • Nostr Map ne stocke que votre clé publique, jamais votre clé privée
  • La connexion utilise une signature cryptographique (NIP-07/NIP-98) — aucun mot de passe n'est jamais transmis
  • Vous pouvez révoquer l'accès à tout moment depuis les paramètres de l'extension

Extension installée ? Si le bouton ne répond pas, rechargez la page.

🖥️

Connectez-vous depuis un ordinateur

La connexion, la modification de profil et la soumission de fiches nécessitent une extension Nostr disponible uniquement sur navigateur de bureau.

Ouvrez nostrmap.fr depuis Chrome, Firefox ou Edge sur votre ordinateur, installez l'extension nos2x ou nos2x-fox, et connectez-vous en un clic.
