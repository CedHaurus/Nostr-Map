Pour ajouter votre profil Nostr à l'annuaire

Connectez-vous avec votre compte Nostr — votre profil sera automatiquement importé et visible dans l'annuaire.

⚡

Se connecter avec Nostr

Nostr est un protocole de réseau social décentralisé. Votre identité vous appartient entièrement — aucun mot de passe, aucun serveur central.

Recommandé — Votre clé privée ne quitte jamais le navigateur. Elle est protégée par l'extension Nostr Map Signer et le code PIN de votre choix.
Pour accéder à votre profil Nostr Map, gérer vos liens sociaux vérifiés ou proposer un utilisateur à l'annuaire, vous avez besoin d'un compte Nostr et de l'extension installée dans votre navigateur.
Compatible avec toute extension NIP-07 : Nostr Map Signer, Alby, nos2x, Amber…
1

Installer l'extension Nostr Map Signer — développée par Nostr Map

L'extension stocke votre clé privée localement dans votre navigateur. Elle signe les requêtes à votre place sans jamais transmettre votre clé à un site externe.

Nostr Map Signer — Firefox

Firefox

Compatible avec : Firefox

1

Installez Nostr Map Signer depuis le bouton ci-dessus, puis cliquez sur son icône dans la barre d'outils Firefox pour ouvrir l'extension.

L'extension Nostr Map Signer affiche votre identité Nostr et votre portefeuille Lightning
2

Si vous avez déjà un compte Nostr, importez votre clé privée (nsec…) dans la section Identité Nostr. Sinon, créez une nouvelle identité directement depuis l'extension.

Votre clé est stockée localement dans votre navigateur. Utilisez le bouton Verrouiller pour la protéger par un code PIN.
3

Une fois votre identité configurée, revenez sur cette page et cliquez sur Se connecter. L'extension vous demandera d'approuver la connexion.

Vous n'avez pas encore de compte Nostr ?

Rendez-vous sur nostr.fr — la meilleure ressource en français pour créer son identité Nostr et découvrir les applications disponibles.

Une fois votre compte créé, revenez ici et importez votre clé privée dans l'extension (étape 1).

Apprendre à utiliser Nostr sur nostr.fr

Se connecter sur Nostr Map

Extension installée et configurée ? Cliquez sur le bouton ci-dessous. L'extension vous demandera d'approuver la connexion.

Votre profil Nostr existant est automatiquement importé sur Nostr Map.

Votre sécurité

  • Votre clé privée ne quitte jamais votre appareil
  • Nostr Map ne stocke que votre clé publique, jamais votre clé privée
  • La connexion utilise une signature cryptographique (NIP-07/NIP-98) — aucun mot de passe transmis
  • Vous pouvez révoquer l'accès à tout moment depuis les paramètres de l'extension

Extension installée ? Si le bouton ne répond pas, rechargez la page.

Solution moins sécurisée — recommandé pour une utilisation ponctuelle. Préférez Nostr Map Signer au quotidien.
Votre clé sert une seule fois à prouver votre identité, puis est effacée. Un jeton de session prend le relais — comme un badge d'accès valable 24h.

Pas encore de compte Nostr ?

Clé nsec — utilisation ponctuelle

Votre clé est utilisée en mémoire pour signer la connexion, puis immédiatement effacée. Elle n'est ni stockée, ni transmise.

Une fois connecté, vous restez identifié pendant 24h ou jusqu'à la fermeture du navigateur.

Pour bénéficier de la protection par code PIN de Nostr Map Signer, connectez-vous depuis Chrome, Firefox, Edge ou Brave sur un ordinateur.
