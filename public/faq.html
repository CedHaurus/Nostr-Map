Tout ce qu'il faut savoir sur Nostr Map et l'écosystème Nostr.

Nostr Map
Qu'est-ce que Nostr Map ?

Nostr Map est un annuaire collaboratif qui recense les utilisateurs francophones du réseau Nostr. Son objectif principal : vous aider à retrouver sur Nostr les personnes que vous suivez déjà sur d'autres réseaux sociaux — Twitter/X, Mastodon, Bluesky, YouTube…

Chaque fiche relie le profil Nostr d'un utilisateur à ses comptes sur les autres plateformes, avec un système de vérification pour garantir l'authenticité des liens. Fini de chercher si votre créateur préféré est sur Nostr : s'il est francophone, il est probablement ici.

Les profils sont ajoutés par soumission communautaire ou par l'équipe éditoriale.

💡 Nostr Map ne stocke aucune clé privée. Votre identité reste sous votre contrôle exclusif.
Qui peut être référencé sur Nostr Map ?

Toute personne active sur Nostr et ayant un lien avec la communauté francophone : créateurs de contenu, développeurs, journalistes, militants, curieux…

Les profils inactifs, les bots ou les profils sans lien avec la francophonie peuvent être refusés ou retirés de l'annuaire.

Compte & Connexion
Comment se connecter à Nostr Map ?

Deux méthodes sont disponibles, sans mot de passe ni email :

🔒 Avec l'extension Nostr Map Signer (recommandé)

  1. Installez Nostr Map Signer sur Firefox ou Chrome / Edge / Brave
  2. Importez votre clé privée (nsec) dans l'extension et protégez-la par code PIN
  3. Cliquez sur "Se connecter" et approuvez dans l'extension

🔑 Avec votre clé nsec directement

  1. Depuis la page de connexion, choisissez l'onglet "Clé Nsec"
  2. Saisissez votre nsec1… — elle est utilisée en mémoire puis immédiatement effacée

Dans les deux cas, vous restez identifié 24h sans avoir à vous reconnecter.

💡 Guide complet avec captures d'écran : page d'aide à la connexion →
Nostr Map accède-t-il à ma clé privée ?

Non, jamais. L'extension Nostr Map Signer agit comme un gestionnaire de clés : elle signe les requêtes localement dans votre navigateur et ne transmet que votre clé publique à Nostr Map.

Votre clé privée (nsec) ne quitte jamais votre appareil.

Comment modifier ma fiche ?

Si votre profil est déjà dans l'annuaire :

  1. Rendez-vous sur votre page de profil
  2. Cliquez sur "Modifier ma fiche" en bas de page
  3. L'extension vérifie que vous êtes bien le propriétaire du compte
  4. Vous accédez à votre tableau de bord pour modifier les informations

Les informations de base (nom, avatar, bio) sont synchronisées automatiquement depuis les relais Nostr. Vous pouvez y ajouter vos liens vers d'autres réseaux sociaux.

Peut-on se connecter depuis un mobile ou une tablette ?

Oui, via votre clé nsec. Depuis la page de connexion, saisissez directement votre nsec1… — elle est utilisée localement pour signer la connexion puis immédiatement effacée de la mémoire.

En revanche, l'extension Nostr Map Signer n'est pas disponible sur mobile : les navigateurs mobiles (Safari iOS, Chrome Android…) n'autorisent pas les extensions. Pour bénéficier de la protection par code PIN et d'une expérience optimale, connectez-vous depuis un ordinateur.

💡 La consultation de l'annuaire fonctionne parfaitement sur mobile sans connexion.
Soumettre & Vérifier
Comment soumettre un utilisateur Nostr ?

Vous devez être connecté pour soumettre un profil :

  1. Cliquez sur Soumettre dans le menu
  2. Entrez le npub de la personne à proposer
  3. Le profil Nostr est récupéré automatiquement depuis les relais
  4. Ajoutez éventuellement ses liens sur d'autres réseaux
  5. Envoyez la proposition — un modérateur l'examinera
💡 Vous ne pouvez pas soumettre votre propre profil via ce formulaire. Connectez-vous directement pour créer votre fiche.
Comment vérifier ses liens réseaux sociaux ?

La vérification prouve que vous contrôlez bien un compte sur un autre réseau. Elle fonctionne via un code de vérification à publier sur le compte concerné :

  1. Connectez-vous et accédez à Mon Profil
  2. Cliquez sur "Ajouter un lien" et choisissez la plateforme
  3. Un code unique vous est fourni — publiez-le sur votre compte (tweet, post…)
  4. Collez l'URL de cette publication et lancez la vérification
  5. Un badge ✓ vert apparaît sur votre profil
Comment signaler un profil ?

Sur chaque page de profil, un lien "Signaler ce profil" est disponible en bas de page.

Motifs acceptés :

  • Usurpation d'identité — le profil se fait passer pour quelqu'un d'autre
  • Demande de retrait — la personne ne souhaite pas être référencée
  • Doublon — ce profil existe déjà sous une autre fiche

Les signalements sont examinés par l'équipe de modération.

Données & Vie privée
Quelles données Nostr Map collecte-t-il ?

Nostr Map ne collecte aucune donnée personnelle sensible. Voici exactement ce qui est stocké :

  • Clé publique (npub) — votre identifiant Nostr, public par nature sur le protocole
  • Nom, avatar, bio, NIP-05 — récupérés automatiquement depuis les relais Nostr publics, modifiables par vous à tout moment
  • Statistiques — nombre d'abonnés et de notes, issus de sources publiques
  • Liens réseaux sociaux — uniquement ceux que vous ajoutez volontairement
  • Horodatages — date d'inscription et dernière connexion

Nostr Map ne collecte ni email, ni mot de passe, ni numéro de téléphone, ni aucune donnée d'identification hors protocole Nostr.

Nostr Map utilise-t-il des cookies ?

Non. Nostr Map n'utilise aucun cookie de traçage, de publicité ou d'analyse.

L'authentification repose sur un token JWT stocké dans le localStorage de votre navigateur — uniquement sur votre appareil, non transmis à des tiers. Ce token expire automatiquement après 24 heures.

Aucun outil de tracking tiers (Google Analytics, Meta Pixel…) n'est intégré au site.

Qui peut voir mes données ?

Toutes les informations affichées sur votre fiche (nom, avatar, bio, liens) sont publiques — c'est le principe même d'un annuaire. Ces données sont d'ailleurs déjà publiques sur le protocole Nostr.

Seule votre adresse IP peut être enregistrée temporairement en cas de signalement, à des fins de protection contre les abus (rate-limiting). Elle n'est pas partagée avec des tiers.

Comment demander la suppression de ma fiche ?

Si votre profil est dans l'annuaire et que vous souhaitez le retirer :

  1. Connectez-vous via votre extension Nostr
  2. Accédez à Mon Profil
  3. Faites défiler jusqu'à la Zone danger en bas de page
  4. Cliquez sur "Supprimer mon profil" et confirmez

La suppression est immédiate et définitive. Vous pourrez vous réinscrire à tout moment.

Si vous n'avez pas accès à votre extension Nostr, utilisez le bouton "Signaler ce profil" avec le motif "Demande de retrait" sur votre page de profil.

Le projet Nostr Map
Nostr Map Signer — l'extension officielle

Nostr Map Signer est l'extension navigateur officielle développée par l'équipe Nostr Map. Elle implémente le standard NIP-07 pour signer vos connexions et interactions Nostr sans jamais exposer votre clé privée.

Elle offre également une intégration portefeuille Lightning via NWC (Nostr Wallet Connect) pour envoyer des zaps directement depuis le navigateur.

  • 🔒 Clé privée stockée localement, protégée par code PIN
  • ⚡ Wallet Lightning intégré (NWC)
  • 🌐 Compatible NIP-07 — fonctionne sur tous les sites Nostr

Navigateurs compatibles :

Chrome Web Store — Chrome · Edge · Brave · Opera Firefox Add-ons — Firefox

Nostr — Le réseau
C'est quoi Nostr ?

Nostr (Notes and Other Stuff Transmitted by Relays) est un protocole de communication décentralisé et résistant à la censure.

Contrairement à Twitter, Mastodon ou Bluesky, Nostr repose sur des clés cryptographiques : votre identité vous appartient entièrement, aucune plateforme ne peut vous bannir ou supprimer votre compte.

Les grands principes :

  • Décentralisé — des milliers de relais indépendants hébergent les messages
  • Résistant à la censure — personne ne contrôle le réseau
  • Souveraineté identitaire — votre clé privée = votre identité
  • Open source — le protocole est ouvert et libre
Comment créer un compte Nostr ?

Créer un compte Nostr revient à générer une paire de clés cryptographiques :

  • nsec — votre clé privée (à garder secrète comme un mot de passe)
  • npub — votre clé publique (votre identifiant, partageable)

Les applications Nostr génèrent ce couple automatiquement à l'inscription. Pour démarrer :

  • Primal — application web et mobile simple pour débuter
  • Damus — application iOS très populaire
  • Amethyst — application Android

Pour tout apprendre sur Nostr en français : nostr.fr →

Comment utiliser Nostr au quotidien ?

Nostr fonctionne comme un réseau social classique, mais vos messages sont publiés sur des relais que vous choisissez. Plusieurs applications accèdent au même réseau :

  • Web — Primal, Snort, Iris
  • iOS — Damus, Primal
  • Android — Amethyst, Primal
  • Desktop — Coracle, noStrudel
💡 Votre clé privée fonctionne sur toutes ces applications — vous n'avez pas besoin de créer un nouveau compte à chaque fois.
