Nostr Map Signer

L'extension navigateur officielle de Nostr Map — signez vos connexions Nostr, gérez vos clés en toute sécurité et connectez votre portefeuille Lightning.

Chrome Web Store Chrome · Edge · Brave · Opera Firefox Add-ons Firefox

Aperçu

Identité Nostr et wallet Lightning
Approbation de connexion NIP-07
Protection par code PIN

Fonctionnalités

🔏

Signature NIP-07

Signez vos connexions et événements Nostr directement depuis le navigateur. Compatible avec tous les clients Nostr majeurs (Primal, Snort, Coracle…).

⚡

Wallet Lightning intégré

Connectez votre portefeuille via Nostr Wallet Connect (NIP-47) pour envoyer des zaps. Compatible Alby Hub, Mutiny, Phoenix.

🔒

Sécurité avancée

Chiffrement AES-256-GCM avec PBKDF2-SHA256 (250 000 itérations). Code PIN ou phrase secrète. Verrouillage automatique après 2h d'inactivité.

🛡

Zéro collecte de données

Vos clés ne quittent jamais votre appareil. Aucun tracking, aucune télémétrie, aucune donnée transmise à des serveurs tiers.

🔑

Gestion des permissions

Contrôlez site par site quelles applications peuvent demander vos signatures. Historique des transactions avec vérification cryptographique.

📱

Écran de verrouillage

Verrouillez l'extension d'un clic. Elle se verrouille automatiquement et demande votre PIN pour reprendre — même si quelqu'un accède à votre navigateur.

Compatibilité

Navigateurs supportés

Chrome Web Store Chrome, Edge, Brave, Opera et tous les navigateurs basés sur Chromium
Firefox Add-ons Firefox

Les navigateurs mobiles (Safari iOS, Chrome Android) ne supportent pas les extensions — connectez-vous depuis un ordinateur pour bénéficier de Nostr Map Signer.
