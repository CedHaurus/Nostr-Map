Ton code challenge — colle-le le temps de vérifier

Une fois le code publié temporairement, entre l'URL de ton profil :

✅ Comment vérifier ?
  1. Choisis la plateforme et clique Obtenir mon code challenge.
  2. Copie le code et colle-le temporairement dans ta bio (ou dans un post pour X).
  3. Entre l'URL et clique Confirmer et vérifier.
  4. Une fois vérifié, tu peux retirer le code de ta bio.
𝕏 Twitter / X — Postez un tweet public contenant le code et collez l'URL du tweet (ex : https://x.com/vous/status/123…).
Bluesky — Collez le code dans votre bio Bluesky, lancez la vérification, puis vous pouvez le retirer. Entrez l'URL de votre profil (ex : https://bsky.app/profile/vous.bsky.social).
YouTube — Collez le code dans la description de votre chaîne, puis entrez l'URL (ex : https://youtube.com/@votre-chaine).
Mastodon — Collez le code dans votre bio, puis entrez l'URL de votre profil sur n'importe quelle instance publique (ex : https://mastodon.online/@vous).
