Relay Nostr Map

Un relay Nostr public francophone, ouvert à tous, sans inscription. Contribuez à la décentralisation du réseau Nostr en l'ajoutant à votre client.

Adresse du relay
wss://relay.nostrmap.net

Comment l'ajouter à mon client Nostr

Dans votre client Nostr, rendez-vous dans les paramètres de relays et ajoutez l'adresse wss://relay.nostrmap.net. Voici les étapes selon votre application :
Primal
Web · iOS · Android
Paramètres → Relays → Add relay → coller wss://relay.nostrmap.net → Confirmer
Damus
iOS
Profil → Relays → icône + → coller wss://relay.nostrmap.net
Amethyst
Android
Paramètres → Network → Relays → Add a relay → coller wss://relay.nostrmap.net
Snort
Web
Paramètres → Relays → champ d'ajout → coller wss://relay.nostrmap.net → Add
Coracle
Web
Menu → Relays → Add relay → coller wss://relay.nostrmap.net
Autres clients
Tous
Cherchez la section Relays ou Serveurs dans les paramètres et ajoutez wss://relay.nostrmap.net.

Caractéristiques

🌐

Ouvert à tous

Aucune inscription, aucun compte requis. Le relay accepte tous les types d'événements Nostr sans restriction.

🛡

Anti-spam

Filtrage des événements via une politique Python pour maintenir la qualité du réseau et limiter les abus.

🔒

TLS automatique

Connexion chiffrée via WebSocket sécurisé (wss://). Certificats TLS gérés automatiquement par Caddy.

👁

Vie privée respectée

Adresses IP anonymisées dans les logs — masque /24 IPv4, /48 IPv6. Aucun profilage des utilisateurs.

⚡

Haute performance

Basé sur strfry, relay écrit en C++ avec architecture LMDB — conçu pour la vitesse et la scalabilité.

🗜

Compression WebSocket

Permessage-deflate activé pour réduire la bande passante et améliorer la réactivité de la connexion.

Stack technique

⚙ strfry (C++ / LMDB) 🔀 Caddy v2 🐍 Python (filtrage) 🐳 Docker Compose 🔒 TLS automatique 🔏 IP anonymisées
